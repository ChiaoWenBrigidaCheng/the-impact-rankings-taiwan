# THE 大學影響力排名（台灣）

此資料視覺化網站可部署於 GitHub Pages。

## 發布方式

1. 建立 GitHub repository。
2. 將本資料夾中的 `index.html` 與 `data.xlsm` 上傳至 repository 根目錄。
3. 到 GitHub repository 的 Settings → Pages。
4. Source 選擇 Deploy from a branch。
5. Branch 選擇 main，資料夾選擇 /root。
6. 儲存後等待 GitHub Pages 產生公開網址。

## 更新資料

未來若有新版 Excel，管理者只要在 GitHub repository 中替換同名 `data.xlsm`，網站重新載入後即可讀取新版資料。前台不提供資料上傳功能。
